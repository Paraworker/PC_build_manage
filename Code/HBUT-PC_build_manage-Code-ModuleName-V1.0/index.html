关于我们

PC构件生产排产系统的简单概述

PC 构件作为装配式建筑的主要组成单元，其生产效率对于推动装配式建筑发展起着重要作用

通过对某 PC 构件厂生产排产系统进行设计：

  • 分析其内容与功能需求
  • 探讨排产系统的层次结构及功能模块，并基于改进后的算法
  • 以生产时间最短为目标，实现不同批次类型构件的合理排产计划

研究表明，PC构件有序排产可以减少各种资源的浪费，降低生产成本，提高构件生产效率，为 PC 构件批量化生产提供参考。

功能

PC 构件生产排产系统是基于 B /S 结构的系统。

项目信息管理

项目信息管理模块分为项目数据列表、项目数据添加和构件批次查看。

构件批次管理

构件批次管理模块分为构件批次列表、构件批次添加和构件类型的查看。

构件类型管理

构件类型管理模块分为构件类型列表、构件类型添加和构件数据查看。

构件数据管理

构件数据管理模块分为构件数据列表、构件数据添加和构件数据修改。

约束组织管理

约束组合管理模块分为约束组合列表和约 束组合添加。
